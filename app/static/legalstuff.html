Open Source Health Check

Legal Information

This tool examines any public GitHub repository and runs a number of different plugins to determine the health of the project. The checks it runs should be used as guidance - we are not responsible for the decisions you make based on this guidance.

This site came about from a session at the Collaborations Workshop 2014, organised by the Software Sustainability Institute. The source code for the site is available at GitHub. This tool created by James Spencer (layout, plugins), Robin Wilson (Flask, plugins), Mark Basham (plugins), Clyde Fare (plugins), Neil Chue Hong (text), and Ling Ge (design). Based on an original idea by Mark Basham, Arfon Smith and James Spencer.

Hosted on Microsoft Azure.
